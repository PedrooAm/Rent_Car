Rent A Car – Sistema de Aluguer de Carros

O Rent A Car é uma aplicação de consola desenvolvida em Python pensada para o consumidor final, oferece uma forma simples, rápida e intuitiva de alugar veículos diretamente a partir do terminal.

Com uma interface baseada em menus interativos, o utilizador consegue criar conta, iniciar sessão e efetuar alugueres de forma prática, sem complexidade técnica.

A Experiência do Utilizador​

O Rent A Car foi desenhada a pensar em quem utiliza o serviço:

  • Processo de utilização simples e guiado
  • Navegação clara através de menus
  • Aluguer rápido de veículos disponíveis
  • Segurança e persistência dos dados
  • Não é necessário conhecimento técnico

Funcionalidades para o Consumidor​

Criação de Conta​

  • Registo rápido e simples
  • Armazenamento seguro dos dados do utilizador

Início de Sessão​

  • Autenticação do utilizador
  • Proteção dos dados de cada cliente

Interface de Utilizador​

A aplicação apresenta menus claros e objetivos:

1 - Criar Conta
2 - Iniciar Sessão
3 - Sair

Como Executar o Projeto​

Pré-requisitos​

  • Python 3.x instalado

Passos​

# 1 - Clonar o repositório
git clone https://github.com/PedrooAm/RentACar

# 2 - Entrar na pasta
cd rent_a_car

# 3 - Executar
python main.py

Tecnologias Utilizadas​

  • Linguagem: Python 3.x
  • Base de Dados: SQLite3
  • Interface: Linha de comandos (CLI)
